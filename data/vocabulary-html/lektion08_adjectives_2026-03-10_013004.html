📚 Vocabulary Reference

spät late ↔ früh (early)

Spanish note: 'tarde' (Spanish) can mean both 'afternoon' and 'late' — in German, 'der Nachmittag' = afternoon, 'spät' = late. Do not confuse. English cognate: none direct. English 'late' is not related to German 'spät'.

frei free / off (not working) / available ↔ besetzt (occupied) / beschäftigt (busy)

English cognate: 'free' — same meaning! 'Ich bin frei' = 'I am free'. Note: German 'frei' can mean both 'free' (available) and 'free' (not working). For a table: 'Der Tisch ist frei' = the table is free/available. For a person: 'Ich bin frei' = I'm free/off work. Spanish: 'libre' (free) is the equivalent — same usage!

✅ Solutions

1. frei
'frei sein' = to be free/off work. Used predicatively after 'sein'.
2. spät
'spät' = late. Used predicatively here with 'ist'. Antonym: früh (early).
3. früh
früh = early, the antonym of spät = late.
4. besetzt
'besetzt' = occupied/taken — antonym of 'frei' for seats/tables.
5. frei
'frei sein' = to be free/available. 'Bist du frei?' is a common A1 question to arrange plans.
6. spät
'Es ist spät' = It is late. Common usage when referring to closing times.
7. frei
'Am Wochenende bin ich frei' — standard expression for being off on the weekend.
8. spät
'so spät' = so late. 'spät kommen' = to come late.
9. frei
'frei sein' used with different days. 'Termin haben' = to have an appointment.
10. spät
'zu spät' = too late. Common expression when missing a time slot.
